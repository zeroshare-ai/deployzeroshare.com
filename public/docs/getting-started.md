# Getting Started with ZeroShare Gateway

Welcome to ZeroShare Gateway! This guide will help you get up and running quickly.

## What is ZeroShare Gateway?

ZeroShare Gateway is an on-premise AI security solution that protects your organization from accidental data leaks when using public AI services like ChatGPT, Copilot, and other AI tools.

## Key Features

- **PII Redaction**: Automatically detects and masks sensitive user information
- **Secret Blocking**: Prevents accidental exposure of API keys and credentials
- **On-Premise Deployment**: Complete control over your data
- **Zero Latency**: Transparent proxy with minimal performance impact

## Quick Start

1. Deploy ZeroShare Gateway from AWS Marketplace
2. Configure your network to route AI API requests through the gateway
3. Start protecting your data immediately
