## Main Characters

### **ALEX CHEN** (Senior DevOps Engineer)
**Physical Description:**
- Age: Early 30s, Asian-American male
- Height: Average (5'9")
- Build: Lean, slightly slouched posture
- Hair: Short, dark, slightly messy, never styled
- Face: Round glasses, slight stubble, neutral expression (rarely smiles, but when he does it's subtle)
- Clothing: Gray hoodie with faded company logo, dark jeans, worn sneakers
- Accessories: Smartwatch (always showing Thursday), multiple keychains with tech stickers
- Posture: Leans back in chair, one hand on keyboard, other holding cold coffee mug

**Facial Expressions:**
- Default: Slight frown, eyes slightly tired but alert
- Amused: One corner of mouth lifts 2mm
- Concerned: Slight eyebrow raise, still neutral
- Satisfied: Eyes crinkle slightly, no smile

**Consistent Details:**
- Desk: Messy but organized chaos, 3 monitors, mechanical keyboard, coffee mug (always full, never drank)
- Background: Code on screens (always Python/DevOps related), terminal windows
- Position: Usually left side of panels, sometimes center

## Office Environment

**Consistent Setting:**
- Modern tech office, open floor plan
- Glass walls, whiteboards everywhere
- Multiple monitors at every desk
- Plants scattered around (some real, some fake)
- Neutral colors: grays, whites, subtle blues
- Accent colors: Purple/indigo (ZeroShare branding) - subtle, in backgrounds

**Key Background Elements:**
- Calendar on wall (always shows Thursday)
- Clocks (all show same time - subtle)
- Company logo (subtle, in backgrounds)
- Security cameras (visible but not obvious)
- Network equipment (in server room, visible through glass)

**Lighting:**
- Natural light from windows
- Overhead fluorescent (soft, not harsh)
- Monitor glow on faces
- Consistent across all episodes

---

## Visual Style Guide

**Art Style:**
- Photorealistic (Google Nano Banana can do this)
- Slightly stylized (not hyper-realistic, but believable)
- Clean lines, clear details
- Professional but approachable

**Color Palette:**
- Primary: Grays, whites, blues (office)
- Accent: Purple/indigo (#667eea to #764ba2) - ZeroShare colors
- Text: Dark gray/black for dialogue
- Backgrounds: Muted, don't compete with characters

**Panel Layout:**
- 3 panels per episode
- Horizontal layout (left to right)
- Consistent spacing
- Clear borders (subtle shadows)

**Text/Dialogue:**
- Bold, readable font
- High contrast (dark text on light backgrounds)
- Speech bubbles: Clean, modern style
- Text size: Readable at small sizes (test on mobile)

---

## Consistency Checklist

For every episode, ensure:
- [ ] Characters look identical to reference
- [ ] Office layout matches previous episodes
- [ ] Lighting is consistent
- [ ] Color palette matches
- [ ] Thursday references visible (calendar, timestamp)
- [ ] Background details reward re-reading
- [ ] Text is readable at small sizes
- [ ] ZeroShare branding subtle but present
