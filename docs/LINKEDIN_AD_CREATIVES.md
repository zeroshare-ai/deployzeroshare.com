## Image Generation Best Practices

### Do's
- Always specify exact dimensions
- Include "professional" and "enterprise" in prompts
- Request "B2B" or "corporate" aesthetic
- Specify "no text" if adding overlays manually
- Use brand colors in prompts when possible

### Don'ts
- Don't use copyrighted logos or brand names
- Don't include recognizable faces (privacy issues)
- Don't create overly complex images (LinkedIn is small)
- Don't use generic stock photo aesthetic
- Don't include competitor branding

### Quality Checklist

Before using any generated image:
- [ ] Correct dimensions for placement
- [ ] Brand colors accurate
- [ ] No artifacts or distortions
- [ ] Works at small thumbnail size
- [ ] Text overlay is readable
- [ ] Professional appearance
- [ ] No copyright concerns
