# Compliance Certification Guides

This folder contains detailed guides for obtaining and maintaining compliance certifications for ZeroShare Gateway.

## Certifications Covered

| Guide | Certification | Target Customers |
|-------|--------------|------------------|
| [soc2.md](soc2.md) | SOC 2 Type II | Enterprise, SaaS buyers |
| [iso27001.md](iso27001.md) | ISO 27001 | Global enterprise, government |
| [gdpr.md](gdpr.md) | GDPR | EU-based organizations |
| [hipaa.md](hipaa.md) | HIPAA | Healthcare organizations |
| [pci-dss.md](pci-dss.md) | PCI DSS | Payment processors, e-commerce |
| [ccpa.md](ccpa.md) | CCPA | California-based businesses |

## How to Use These Guides

Each guide follows a consistent structure:

1. **Overview** - What the certification is and why it matters
2. **Requirements** - What you need to demonstrate
3. **Application Process** - Step-by-step instructions
4. **Evidence Needed** - What documentation to prepare
5. **CI/CD Integration** - Automated evidence generation
6. **Costs & Timeline** - Budget and schedule planning
7. **Maintenance** - Ongoing requirements after certification

## Templates

The `templates/` folder contains reusable policy templates:

- Security policies
- Incident response plans
- Data processing agreements
- Risk assessments
- Training documentation

## Evidence Collection

The `evidence/` folder contains guides for collecting specific types of evidence:

- Technical controls documentation
- Administrative controls documentation
- Physical controls documentation

## Quick Start

1. Identify which certifications you need based on your target customers
2. Read the corresponding guide(s)
3. Complete the prerequisites checklist
4. Engage with an auditor or certification body
5. Use CI/CD to generate ongoing evidence
